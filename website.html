Attivazione Altervista (Alterblog)

    N.B.: Esistono 2 versioni di Altervista, Altervista Hosting e Alterblog.

    • Altervista Hosting: I vantaggi che offre sono un Web Server con database MySQL dove potete caricare i vostri file HTML, CSS, JAVASCRIPT, PHP ecc.
      Molto utile se avete dei siti web creati da voi (come questa guida). Lo svantaggio (che io ritengo molto vantaggioso) è che dovete costruirvi da zero WordPress. Scaricando da WordPress i file necessari e confiurali con il database.
    • Alterblog (scelta che utiliziamo per questa guida): I vantaggi che offre è l'intera suite di WordPress già installato e confiurato (pronto al uso).
      Lo svantaggio è che non potete modificare il database e non potete caricare siti web cretati HTML ecc. Tutto si basa su WordPress.
    Entrambi le versioni di Altervista sono GRATUITE ma richiedono almeno 2 GIORNI LAVORATIVI PER L'ATTIVAZIONE DEL ACCOUNT.

  1. Per attivare Altervista recarsi sul sito web di Altervista e premere su PROSEGUI >.
  2. Vi verrà richiesto di inserire un nome per il dominio.
    Potete inserire il vostro nome cognome oppure il nome della azienda dove state facendo il sito Web.
  3. A questo punto vi chiede di inserire i vostri dati personali.
    La motivazione dei 2 giorni di verifica è dovuta da un CONTROLLO UMANO quindi non potete mettere dati finti.
    L'unica cosa che potete mettere sono i VOSTRI VERI DATI REALI.
    Quello che potete mettere finto è la data di nascita se volete potete mettere quella reale.
  4. A fine processo se tutto va a buon fine vedrete questa schermata.
    Riceverete una email entro 48 con le credenziali di accesso al portale di Alterblog.
  5. Ora puoi usufruire WordPress Online con Alterblog
  6. CONSIGLIO: Inizia ad installare (se non presente) il plugin di Backup e Ripristino per WordPress trovi una guida QUI
